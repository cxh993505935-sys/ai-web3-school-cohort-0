# AI × Web3 School 学习计划

## 学习目标

### 总体目标
- 掌握 AI 和 Web3 核心技术
- 能够独立开发 AI × Web3 项目
- 参与黑客松并取得成果
- 建立个人技术品牌

### 具体目标
1. **AI 方向**: 从有基础到熟悉，掌握主流 AI 框架和应用
2. **Web3 方向**: 从新手到有基础，理解区块链原理和智能合约开发
3. **开发能力**: 从 C++ 后端到全栈开发
4. **项目经验**: 完成至少 1 个黑客松项目

## 学习时间安排

- **每日投入**: 约 2 小时
- **学习时段**: 根据个人时间灵活安排
- **每周目标**: 完成 1-2 个学习任务

## 学习路径

### 第一周：环境搭建与基础
- [x] GitHub CLI 安装与配置

- [ ] AI 核心概念复习

### 第三周：Web3 基础学习
- [ ] 区块链原理学习

### 第四周：项目实践
- [ ] AI × Web3 项目构思
- [ ] 原型开发
- [ ] 黑客松准备
