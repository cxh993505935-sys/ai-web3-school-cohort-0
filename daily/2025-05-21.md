## 🔧 踩坑记录

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| `pip install` 报错 externally-managed | Debian/Ubuntu PEP 668 保护系统 Python | 使用 `uv venv` 创建独立环境 |
| Python 3.11 运行 cli-hub 报 SyntaxError | f-string 不支持反斜杠（3.12+ 才支持） | 换用 Python 3.12 |
| `python3.12 -m venv` 失败 | 缺少 python3.12-venv 包，无 sudo 权限 | 用 `uv venv --python 3.12` 代替 |
